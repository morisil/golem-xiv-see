## Why?

A page that looks obvious to a human can be invisible, ambiguous, or actively
misleading to an AI agent driving a browser. `golem-xiv-see` exposes the web the
way an agent perceives it — the structure, text, and affordances it can actually
reason about — so you can debug, design, and build agentic browsing on top of a
faithful picture of what the model sees.
